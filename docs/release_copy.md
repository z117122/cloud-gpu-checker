## Release Title

`v0.1.0 - First public release`

## Release Notes

First public release of Cloud GPU Checker.

This version includes:

- SSH-based remote experiment inspection
- CLI checker
- Windows GUI checker
- Multiple saved instance profiles
- Progress summary from logs and result folders
- Rough ETA for the current run and the whole batch

## Social Copy

I open-sourced a small tool I built for checking real experiment progress on remote GPU servers.

It goes beyond `nvidia-smi` and tries to answer the questions you actually care about during long training jobs:

- what is running now
- which model and horizon are running
- what has already finished
- whether `MSE / MAE` already appeared
- how much time is probably left
